# The New AutoHotkey Pastebin (yay)

## Project Setup

1.  Install [Bun](https://bun.sh) any way you like
2.  `cd` to the repo root (where `package.json` is)
3.  Install all dependencies with `bun install`

### Development

4.  Start development server with `bun dev`

## Deployment

4.  Build project for production with `bun run build`
5.  Run production server with `bun run build/index.js`
